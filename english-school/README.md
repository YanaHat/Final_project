# English Prime 🇬🇧

Сайт школы английского языка в Вроцлаве. Проект реализован на **React + TypeScript + Vite**.

## Функционал
- **Курсы**: Информация об обучении для взрослых, разговорных клубах и индивидуальных занятиях.
- **Новости**: Актуальные события школы (используется имитация API сервиса).
- **FAQ**: Ответы на часто задаваемые вопросы с интерактивным интерфейсом.
- **Тематизация**: Поддержка светлой и темной тем оформления.
- **Форма заявки**: Удобная форма записи на пробное занятие.

## Технологии
- **React 18**
- **TypeScript** (строгая типизация компонентов)
- **Context API** (управление глобальным состоянием темы)
- **CSS3** (адаптивная верстка, CSS переменные)

## Установка и запуск

1. Склонируйте репозиторий:
  git clone [https://github.com/ваш-логин/english-school.git](https://github.com/ваш-логин/english-school.git)
2. Установите зависимости:
  npm install
3. Запустите проект в режиме разработки:
  npm run dev
